Orb Engine (Ongoing Project)


Description: An OpenGL/SDL2 C++ game engine project where most of my focus will be recreating game engine features, mainly rendering. This is a passion project I am continuing to work on and will continue to do so as I learn more about game engines. A lot of it is inspired through online resources and from having worked with Unity.

List of implementations:

  • Entity Component System | List of Components Supported Below
    • Camera
    • Mesh Renderer
    • Lights: Directional, Spot, Point
  • Editor for Managing Assets and Game Objects
  • 3D Math Library
  • Transforms
  • Lighting
  • Materials
  • Post Processing
  • Skybox
  • Model loading
  • Deferred Rendering

Team size: Solo
Engine: Custom C++ 3D Engine (Visual Studio 2022)
Platform: Windows
Shader Language: GLSL
